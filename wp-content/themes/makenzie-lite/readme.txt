== Description ==
Makenzie Lite is a dynamic and visually stunning Free WordPress Theme specially designed for Handmade Shops, Magazine, Craft Blogs, Fashion & Lifestyle Blogs. The theme is easy to set up and customize, you need just a few clicks, and you can entirely change the look of the website!

== Easy to install ==
The installation process is quite simple and easy, it takes just a few minutes!

Activate the theme and go to `Appearance > Customizer`.

## Toolbar settings
All options from this section will appear to the toolbar

## Header Options & Colors
Use this section to set the header style

##  Home - Promo Boxes Module
This option will take effect if you use a static page, and make sure the static page is using the "Home" Template

## Home - Intro Box Module
This option will take effect if you use a static page, and make sure the static page is using the "Home" Template

## Social Icons
Add social Icons which will be used for toolbar and header area

## Single Post Layout Options
A few options for Single Post Layout

## Footer Options
Option to add the Copyright

## The theme has 3 custom templates:
1. Home page template - To display Promo Boxes and Intro box module
2. Blog - To display blog posts.
3. Full Width - to display full width page
